Store Sales Analysis Project
Overview

This project is a sales data analysis for a retail store using Power BI. The goal is to understand sales trends, identify top-performing products, and provide actionable insights to improve business performance.

Tools & Technologies

Power BI: Data visualization and dashboard creation.

Power Query: Data cleaning, transformation, and preparation.

Excel/CSV: Source data files.

Features / Dashboard Highlights

Sales Overview: Total sales, revenue trends, and growth analysis.

Product Analysis: Top-selling products and categories.

Regional / Store Analysis: Sales performance by location or branch.

Interactive Filters: Slicers for time periods, product categories, and store locations.

Visual Insights: Charts, tables, and KPIs to track key metrics.

How to Use

Open the Power BI file (.pbix).

Use the filters to explore sales data interactively.

Analyze visualizations to uncover sales patterns and insights.
