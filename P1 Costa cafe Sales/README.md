# Costa Coffee Sales Dashboard

## Project Overview
This project is an Excel dashboard analyzing sales data for Costa Coffee. 
It includes sales trends, product performance, and insights for business decisions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Graphs

## Dataset
The dataset includes daily sales, product categories, and store locations.

## Features
- Interactive Pivot Tables
- Sales Trends Charts
- Product Performance Analysis

## How to Use
- Open the Excel file
- Navigate between sheets for Sales Overview, Charts, and Insights
