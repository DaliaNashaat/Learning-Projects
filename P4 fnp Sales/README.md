fnp for Presents Sales Analysis Project
Overview

This project is a sales data analysis for FNP for Presents using Excel, Power Query, and Power Pivot. The goal is to clean, transform, and analyze sales data to identify trends, top-selling products, and actionable business insights.

Tools & Technologies

Excel: Data storage, organization, and basic analysis.

Power Query: Data cleaning, transformation, and preparation.

Power Pivot: Data modeling, creating relationships, and performing calculations with DAX.

Features / Analysis Highlights

Sales Overview: Total sales, revenue trends, and growth analysis.

Product Performance: Top-selling products and categories.

Branch / Regional Analysis: Insights by location or store.

Customer Insights: Analysis of purchasing behavior.

Data Modeling: Using Power Pivot to relate tables and calculate KPIs.

Pivot Tables & Charts: Interactive summaries for easy exploration of data.

How to Use

Open the Excel file(s) with Power Query and Power Pivot enabled.

Explore the pivot tables and charts to analyze sales trends.

Use slicers and filters to focus on specific products, time periods, or locations.
